raw-sockets

Description:

Simple access to UNIX raw sockets.

Author:

Ben Kurtz.

Version:

  • 0.1 Initial version.

Requires

  • srfi-66

Usage:

(require-extension raw-sockets)

Download:

raw-sockets.egg

Documentation:

This egg provides the ability to open, send, and close packet sockets.

Packet sockets are even less cooked than "raw" sockets, letting you set layer 2 fields manually.

Currently, this egg has only been ported to OSX and Linux.

Future Work: port to Win32, make work with more than one interface at a time.


(procedure) (raw-open INTERFACE)

Opens a packet socket on the given interface name.

(procedure) (raw-send BUFFER LEN)

Sends a buffer of bytes out the opened socket. BUFFER is the u8vector containing the data to be sent, LEN is the length of BUFFER in octets.

(procedure) (raw-close)

Closes the packet socket.
