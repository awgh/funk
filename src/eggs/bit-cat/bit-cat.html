bit-cat

Description:

Bitwise Concatentor for u8vectors.

Author:

Ben Kurtz.

Version:

  • 0.1 Initial version.

Usage:

(require-extension bit-cat)

Download:

bit-cat.egg

Documentation:

This egg provides a way to concatenate two u8vectors on a bitwise basis.

Pass in a left-hand-side (LHS) value and a right-hand-side (RHS) value and bitlengths for both.

The bits of the LHS will be concatenated into the RHS in a right-justified manner.

This is mainly useful for aggregating bitfields.

Future Work: Also supply left-justified concatenation.


(procedure) (bit-cat LHS LHS-BL RHS RHS-BL)

Concatenate LHS-BL bits from LHS into u8vector RHS in a right-justified manner. Resulting concatenated vector will be in RHS, result is the total resulting bitlength. RHS-BL is the bitlength of data already used in RHS.
